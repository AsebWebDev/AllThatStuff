# AllThatStuff
multi-player browser word guessing game

## Features
* Players take turns
* Select and drag&drop elements to make the other players guess a word
* Live update of moving elements for all players

## Technologies
HTML5, CSS3, JavaScript, jQuery, Socket.IO
